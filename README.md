使用教學:
1. 到這個網址https://github.com/JACK900727/Eprofolio-robotdog/releases
2. Asset欄位點開，下載裡面名為default.zip的檔案
3. 解壓縮
4. 點進去之後執行有一個檔案叫做<幾田的機器狗.exe>
5. 會跳出一個通知說:windows已保護您的電腦
6. 點其他資訊
7. 點仍要執行
8. 輸入帳號密碼後點按鈕
9. 會開啟一個chrome視窗 此時可以切回去原本那個程式追蹤填寫進度
